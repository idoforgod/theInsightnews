
# Project Overview: The Insight Chronicle

## 1. 프로젝트 소개
**The Insight Chronicle**은 실시간 검색 데이터와 Google Gemini AI의 추론 능력을 결합하여, 단순한 정보 전달을 넘어선 '통찰(Insight)'과 '미래 시나리오'를 제공하는 지능형 뉴스 플랫폼입니다. 사용자는 키워드를 입력하여 즉석에서 심층 기사를 생성하거나, 관리자가 큐레이션한 기사를 신문 형태의 UI로 소비할 수 있습니다.

## 2. 핵심 가치 (Core Values)
1.  **Real-time Intelligence**: 현재 시각 기준 최신 속보를 수집하여 기사화.
2.  **AI Insight & Inference**: 단순 사실 나열이 아닌, 전문 용어 해설(Insight Box)과 미래 시나리오(The Insight Engine) 제공.
3.  **Journalistic Aesthetics**: 종이 신문의 질감(`newspaper-shadow`)과 레이아웃을 웹에서 완벽하게 구현.
4.  **Crisis Tracking**: 이슈별 타임라인을 통해 위기의 흐름을 추적(Timeline Widget).

## 3. 서비스 슬로건
> "전문 미래학자(Professional Futurist) 최윤식 최현식. 지식과 미래 통찰의 융합"

## 4. 주요 타겟 사용자
*   최신 경제, 금융, 기술 트렌드를 심층적으로 파악하고자 하는 전문가 및 일반 독자.
*   단편적인 뉴스가 아닌, 사건의 인과관계와 미래 파장을 알고 싶은 사용자.
